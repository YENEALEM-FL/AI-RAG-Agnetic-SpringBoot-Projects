## Read ME!
This repository contains a number of startup AI artifacts that are implemented using:

    1.  SpringBoot
    2.  SpringAI
    3.  Chat APIs(Open AI, Claudi, Gemini, and Others)
    4.  Several Vector Databases(Oracle23, Postgres, Opensearch, ElastiSearch, and other vector databases)

To run the artifacts:
- clone a particular repository
- use your own API-KEY in places of the keys requirement.
